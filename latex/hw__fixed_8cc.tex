This function calculates the standard deviation of an array. 

\hypertarget{main__broken_8cpp_autotoc_md0}{}\subsection{How this function calculates standard deviation}\label{main__broken_8cpp_autotoc_md0}
Standard Deviation Equation

$ SD = \sqrt{\frac{\displaystyle\sum_{i=0}^{n} | a[i] - \mu |^2 }{n}} $


\begin{DoxyEnumerate}
\item Find the mean (mean = $ \mu $) of all elements in array a using n as the base
\item For each member in a, find the square of its difference from the mean (example below).
\begin{DoxyItemize}
\item $ \mu $ = 2
\item a\mbox{[}1\mbox{]} = 5
\item Equation\+: $ (a[1] - \mu)^2 = (5-2)^2 = 9 $
\end{DoxyItemize}
\item Sum all the values from step 2
\item Find them mean of step 3 using n as the base
\item Take the square of step 4
\end{DoxyEnumerate}
